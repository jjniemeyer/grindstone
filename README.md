# Grindstone

A pomodoro timer TUI application with session tracking and statistics visualization, built with Ratatui.

This is primarily an experiment with Claude Code and Rust. If you like it, fork it and do whatever you want.

## Features

- Pomodoro timer with configurable work/break durations
- Session tracking with custom categories
- Statistics visualization by category
- Persistent storage with SQLite

## Build

```bash
cargo build                  # Debug build
cargo build --release        # Release build (optimized for size)
```

## Run

```bash
cargo run
```

## Test

```bash
cargo test
```
